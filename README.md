# API endpoints
1. Add /add/:num1/:num2 (GET)
2. Subtract /subtract/:num1/:num2 (GET)
3. Multiply /multiply/:num1/:num2 (GET)
4. Devide /devide/:num1/:num2 (GET)
5. Factorial /factorial/:num1 (GET)
# Running the app
## running the CLI app
- `perl cal.pl`
## running the web app
- `perl cal_web.pl`
## running the test
- change into t directory and run `perl test_cal.t`
